Android users can install Speechify using the Google Play Store.

#. Open the Google Play Store.
#. Search for "Speechify."
#. Select the official Speechify app.
#. Tap the Install button.
#. Wait for installation to finish.
#. Open the application.
#. Log in or register.

How to Install Speechify on iPhone
==================================

iPhone users can install Speechify from the Apple App Store.

Steps:

#. Open the App Store.
#. Search for "Speechify."
#. Select the official application.
#. Tap Get to download.
#. Complete installation.
#. Launch Speechify.
#. Sign in to begin using the app.
